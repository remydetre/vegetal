# MODULE "Bouton ajouter des quantités depuis les listes produits"
#
# Disponible en version 1.6 et version 1.7
# Aucun prérequis technique nécessaire
#
# Installation, 
# Veuillez vous assurer de disposer d'une version 1.6 au moins (Vérifiez dans votre back office - en haut de l'écran, à côté du logo prestashop logo ou dans Menu / Paramètres avancés / Informations)
# 
# Télécharger votre zip via le Menu / Modules => bouton "Ajouter un nouveau module" (suivez les recommandations Prestashop)
# Une fois le zip uploadé sur votre serveur, cliquez sur Installer, puis Configurer.
# 
# Une fois le module “installé” & ”activé”, le bouton “ajouter au panier” et l’input quantité sont affichés automatiquement sur la liste produit de votre site web.
# Le module tient compte de vos stocks en back office.
#
# Le module peut être “désactivé” : le bouton disparaît de la vignette produit sur les listes catalogues.
